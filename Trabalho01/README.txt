Este zip contêm 4 arquivos:

- Trabalho01.ipynb: arquivo do tipo jupyter-notebook com o relatório juntamente ao código referente ao trabalho
- Trabalho01.py: arquivo contendo somente o código Python do notebook para execução individual
- requirements.txt: arquivo contendo todas as dependências necessárias para executar o código do arquivo acima
- Trabalho01.pdf: conversão do arquivo notebook para o formato PDF

Para visualizar e executar o notebook, acesse https://colab.research.google.com, clique em "upload" e insira o arquivo "Trabalho01.ipynb"

Para executar o código Python, execute "pip install --user -r requirements.txt" e, após instalação das dependências, execute "python Trabalho01.py"
Obs: em algumas distribuições é necessário utilizar "pip3" e "python3" para a correta execução.
